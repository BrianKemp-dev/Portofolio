Test je kennis over architectuur en security!

1. Wat betekent two factor authentication?

  • A) Een methode waarbij je je wachtwoord moet invoeren op twee verschillende apparaten voordat je kunt inloggen.
  • B) Een beveiligingsmethode waarbij je twee verschillende verificatiestappen moet doorlopen om toegang te krijgen tot je account.
  • C) Een systeem waarbij je elke keer een nieuw wachtwoord krijgt voor elke loginpoging.

2. Wat is een SSL-certificaat?

  • A) Een document dat bevestigt dat een website veilig is voor gebruik, maar geen versleuteling biedt.
  • B) Een beveiligingstechnologie die zorgt voor een versleutelde verbinding tussen een webbrowser en een webserver, zodat gegevens veilig kunnen worden uitgewisseld.
  • C) Een certificaat dat nodig is om een website te registreren bij een domeinnaamprovider.

3. Wat zijn ethische hackers?

  • A) Hackers die zonder toestemming systemen aanvallen om persoonlijk voordeel te behalen.
  • B) Hackers die software maken om de beveiliging van internetverbindingen te verbeteren.
  • C) Beveiligingsexperts die kwetsbaarheden in systemen zoeken en oplossen met toestemming van de eigenaar om de veiligheid te verbeteren.

4. Wat is het verschil tussen symmetrische en asymmetrische encryptie?

  • A) Symmetrische encryptie maakt gebruik van twee sleutels, terwijl asymmetrische encryptie één sleutel gebruikt voor zowel encryptie als decryptie.
  • B) Symmetrische encryptie gebruikt één sleutel voor zowel encryptie als decryptie, terwijl asymmetrische encryptie twee verschillende sleutels gebruikt: een openbare sleutel en een privésleutel.
  • C) Symmetrische encryptie is langzamer dan asymmetrische encryptie, omdat het meer rekencapaciteit vereist.

5. Wat is een DDoS-aanval?

  • A) Een aanval waarbij meerdere computers tegelijk proberen de toegang tot een website of netwerk te blokkeren door het systeem te overspoelen met verkeer.
  • B) Een techniek waarbij hackers de identiteit van een legitieme gebruiker overnemen om toegang te krijgen tot vertrouwelijke gegevens.
  • C) Een type aanval waarbij de hacker probeert in te breken in een systeem door gebruik te maken van kwetsbaarheden in de software.

6. Uit welke lagen bestaat het drielagenmodel?

  • A) Fysieke laag, Logische laag, Toepassingenlaag.
  • B) Netwerklaag, Applicatielaag, Dataopslaglaag.
  • C) Hardwarelaag, Softwarelaag, Netwerklaag.

7. Wat is het verschil tussen een emulator en een virtual machine?

  • A) Een emulator is een software die meerdere besturingssystemen tegelijk kan draaien, terwijl een virtual machine slechts één besturingssysteem kan draaien.
  • B) Een emulator draait op hardware en simuleert een volledig ander besturingssysteem, terwijl een virtual machine een volledig besturingssysteem draait binnen een bestaand besturingssysteem.
  • C) Een emulator draait alleen op een virtuele omgeving, terwijl een virtual machine altijd fysieke hardware nodig heeft om te draaien.

8. Wat is het verschil tussen een IPv4 en een IPv6 adres?

  • A) IPv4 maakt gebruik van hexadecimale getallen, terwijl IPv6 decimale getallen gebruikt voor adressen.
  • B) IPv4 ondersteunt alleen privé netwerken, terwijl IPv6 alleen voor openbare netwerken wordt gebruikt.
  • C) IPv4-adressen zijn 64-bits en IPv6-adressen zijn 128-bits, wat zorgt voor meer unieke adressen in IPv6.

9. Wat is DNS en waarvoor wordt dit gebruikt?

  • A) DNS (Domain Name System) is een systeem dat domeinnamen vertaalt naar IP-adressen, zodat computers websites kunnen vinden.
  • B) DNS is een protocol voor het verzenden van e-mailberichten tussen computers op internet.
  • C) DNS is een beveiligingsprotocol dat wordt gebruikt om gegevens te coderen tijdens het verzenden over internet.

10. Wat betekent de Von Neumann-architectuur?

  • A) Een type netwerkarchitectuur voor het verbinden van computers in een lokaal netwerk.
  • B) Een type architectuur waarbij de gegevens en instructies in verschillende geheugens worden opgeslagen en uitgevoerd.
  • C) Een type computerarchitectuur waarbij gegevens en instructies in hetzelfde geheugen worden opgeslagen en uitgevoerd.
